# WashPanda

WashPanda is organized as a client/server project.

## Structure

- `client/` — React, Vite, and Tailwind frontend
- `server/` — Express API scaffold

## Commands

Install dependencies separately in each application:

```bash
cd client && npm install
cd ../server && npm install
```

From the repository root:

```bash
npm run client
npm run server
npm run build
npm start
```
